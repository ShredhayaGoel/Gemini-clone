# Gemini Clone (React)

This project is a **Gemini AI Clone** built using React.  
It replicates the basic UI and functionality of the Google Gemini chatbot interface.

Users can enter prompts and receive AI-generated responses in a clean and responsive chat interface.

## 🚀 Features

- Modern chat interface
- Prompt input and response display
- Loading animation
- Responsive design
- React component-based architecture

## 🛠️ Technologies Used

- React
- JavaScript
- HTML5
- CSS3
- AI API Integration
